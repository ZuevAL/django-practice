# Изучение Django: от основ до сложных связей в ORM

В этом репозитории собраны учебные проекты, в которых я отрабатывал ключевые концепции Django: работу с шаблонами, построение реляционных связей в БД и реализацию CRUD-функционала.

## Содержание

### 1. Основы и работа с шаблонами (`01_Basics_and_Templates`)
* **Фокус:** Наследование шаблонов, организация статики и маршрутизация.
* **Стек:** Django, HTML/CSS.

### 2. Глубокое погружение в Django ORM (`02_ORM_Relationships`)
Самый важный блок, где реализованы все типы реляционных связей:
* **One-to-One:** Реализация уникальных связей между моделями.
* **One-to-Many (ForeignKey):** Построение иерархических структур данных.
* **Many-to-Many:** Работа со сложными связями и промежуточными таблицами.
* **ORM Methods:** Отработка базовых методов фильтрации и получения данных из БД.

### 3. Реализация CRUD-интерфейса (`03_CRUD_Operations`)
* **Задача:** Создание системы управления данными (Create, Read, Update, Delete).
* **Ключевые навыки:** Работа с формами (`ModelForm`), обработка POST-запросов и редиректов, интеграция логики удаления и редактирования объектов в шаблоны.# django-practice
